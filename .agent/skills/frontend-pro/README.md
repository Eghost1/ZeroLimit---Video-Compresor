## Descripción y Propósito

**Frontend Pro** instruye al agente para construir interfaces web inolvidables y fuera de lo ordinario. Su principal consigna es evitar estéticas genéricas ("AI slop") mediante la toma deliberada de decisiones de diseño creativo (minimalismo táctil, maximalismo ruidoso, brutalismo, retro-futurismo). Combina una codificación moderna de frontend (HTML/CSS o frameworks como React y Vue) con un enfoque profundo en el "Design Thinking", animaciones sofisticadas, paletas audaces y tipografía expresiva.

## Funcionalidades Principales

- **Despliegue Estético Audaz**: Identifica un "tono" coherente y lo ejecuta hasta el límite, alejándose de los patrones estándar grises y aburridos.
- **Microinteracciones y Animaciones (*Motion*)**: Uso meticuloso de transiciones de revelaciones escalonadas (*animation-delay*), decoraciones activadas al *hover* y paralajes limpios controlados por las interacciones del cursor.
- **Tipografía Altamente Curada**: Recomienda escapar de "fuentes seguras" acoplando familias tipográficas de titular (*display*) muy decorativas con tipografías ultra legibles de párrafo, configuradas matemáticamente a nivel de espaciado.
- **Ingeniería Espacial**: Crea layouts de asimetría controlada o densidades matemáticas meticulosas (apilamiento orgánico en contra o apoyando sistemas *Grid* rígidos).
- **Procesamiento de Atmósferas**: Abandona el fondo sólido simple; usa sombras dinámicas, de gradientes fluidos (*Mesh Gradients*), efectos de refracción (*Glassmorphism*) o texturas granosas/ruido (*grain*).

## Casos de Uso

### Caso 1: Dashboard Minimalista de Alta Precisión

* **Contexto**: Se requiere una interfaz corporativa de analítica pesada.
- **Acción**: En lugar de "otro framework estándar de Bootstrap", el agente opta por un refinamiento minimalista suizo. Utiliza colores de borde muy imperceptibles, densidad tipográfica reducida, grandes márgenes (uso de espacios negativos) e implementa tablas con transiciones ultrarrápidas y sutiles.

### Caso 2: Landing Page Disruptiva / Portafolio "Art-Deco"

* **Contexto**: Un estudio de ilustración pide un sitio impactante para destacar sus dibujos.
- **Acción**: Ejecución bajo la temática Brutalista/Maximalista. El frontend empuja la asimetría, la ausencia de márgenes canónicos, colores primarios muy estridentes en hover, y una barra de progreso de scroll invertida o caótica para lograr la dirección creativa esperada.

### Caso 3: Interacción de Tarjetas de Precio

* **Contexto**: Las típicas 3 tarjetas de *pricing tier* (Basic, Pro, Enterprise).
- **Acción**: Aplicación de animaciones elegantes donde el paso del mouse emite destellos fluidos bajo la tarjeta (efecto luz nocturna), el fondo oscuro presenta un mesh gradient lento y la tarjeta central tiene un desfase de altura diagonal destacada.

## Requisitos o Directrices Operativas

- **No plantillas genéricas**: No se deben generar contenedores típicos y predecibles; cada implementación debe ser estéticamente particularizada y reflexiva.
- Utilizar variables nativas en CSS exhaustivamente.
- Refinar de manera excepcional los aspectos de tipografía y paleta, sin recurrir a contrastes arbitrarios sin justificación cromática.
- La ejecución del CSS (o sistema de diseño asociado como Tailwind/Styles components) debe resultar en un producto "listo para producción".
